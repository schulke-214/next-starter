# Button

With `MDX` we can define a story for `Button` right in the middle of our
markdown documentation.

<Preview>
	<Story name='with text'>
		<Button disabled={boolean('Disabled', false)} onClick={action('click')}>
			{text('Label', 'Hello Storybook')}
		</Button>
	</Story>
</Preview>
